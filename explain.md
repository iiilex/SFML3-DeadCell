# 我需要什么呢?

## 一套物理系统

包括了在这个游戏世界中的各种物理模型

0. 世界中的物理参数
   1. 重力加速度
   2. 摩擦系数

1. 一套集成的物理参数 PhysicsData
   1. 世界中的坐标 (左上为(0,0), 向右为x,向下为y)
   2. 速度(x方向, y方向)
   3. 碰撞箱
   4. 是否撞墙
   5. 是否落地
   
2. 一套物理世界的判断更新规则 PhysicsSystem
   1. 根据重力加速度更新y方向速度
   2. 按照速度更新当前坐标
   3. 修正坐标(假如撞墙/撞到实体)
   4. 判断两个碰撞箱是否接触
   5. 判断当前物体是否着地(目前是简单版)
   6. 判断当前物体是否撞墙(目前是简单版)



## 一个角色 Player 类
1. 包含什么参数呢?
   1. 一个PhysicsData类型的参数myData，代表自身物理数据
   2. vector\<Animation> myAnimation 存储角色包含的动作序列
   3. 角色的血量
   4. 角色当前应当进行的动作序列
   
2. 初始化
   1. 初始化myData等物理数据
   2. 初始化myAnimation, 定义各个动作
   3. 初始化其他东西

3. 接受键盘的操作(压入动作序列)
   1. 跑步
   2. 跳跃
   3. 翻滚
   4. 下蹲

4. 更新当前状态(暂时没有加入动画)
   1. 按照队首状态更新物理数据
   2. updatePosition()
   3. 队首状态更新动画系统(过一会写)
